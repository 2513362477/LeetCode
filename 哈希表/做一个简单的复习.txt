1. 哈希表（散列表）hash table
一般哈希表都是用来快速判断一个元素是否出现集合里
2. 哈希函数 hash function
哈希函数，把学生的姓名直接映射为哈希表上的索引，然后就可以通过查询索引下标快速知道这位同学是否在这所学校里了。本质是将一个大集合映射到小集合上。
3. 哈希碰撞（哈希冲突）
不同元素访问了相同下表。解决方法：链表法（常用）、线性探测法
4. 常见的哈希结构（in java）
数组、集合set、映射map
hash map, hash set, tree map
